This repo provides JavaScript and Python solutions for Coderbyte challenges. It’s designed to help you ace coding interviews on coderbyte.com. 

## Key Features
- Solutions categorized by difficulty: easy, medium, hard.
- Unit tests using Jest (JavaScript) and pytest (Python).
- A `generate.js` script to create challenge templates effortlessly.

## Quick Start

### Prerequisites
- Node.js and npm
- Python 3 and pip3

### Installation

Clone the repo:

```bash
git clone https://github.com/justin-napolitano/coderbyte-python-javascript
cd coderbyte-python-javascript
```

Install dependencies:

```bash
npm install
pip3 install pytest
```

### Running Tests

For JavaScript:

```bash
npm test
```

For Python, navigate to the difficulty folder and run:

```bash
pytest
```

### Gotchas
- Make sure you're in the correct folder for Python tests.
- Enhance `generate.js` for better template handling later.
